for循环语法组成部分：

  1. 创建循环初始值
  2. 设置（验证）循环条件
  3. 条件成立执行循环体内容
  4. 当前循环结束执行步长累计操作

传统基于DOM方式实现业务需求的过程：

  1. 想操作谁就先获取谁
  2. 给某元素绑定某事件
  3. 在事件触发的时候修改元素的样式或内容

VUE和React的方法：

  1. 先准备出需要的视图（页面，效果）
  2. 准备数据
  3. 让视图和数据之间建立关联
  4. 只需改变数据，框架就会完成剩下操作

所以是一种基于数据的全新思想

所以操作思想分为

  1. 操作DOM
    • 直接操作DOM
    • JQuery
  2. 操作数据
    • VUE, React

形参默认值处理

  1. 数据比较

    if([n] === undefined){
    [n] = 0;
}

  2. 数据类型检测

    if(typeof [m] === 'undefined'){
    [m] = 0;
}

    💡'undefined'是因为typeof检测结果返回的都是string字符串类型

匿名函数

匿名函数表达式: 把一个匿名函数本身作为赋值给其他东西, 这种函数一般不是手动触发执行, 而是依靠其他程序触发执行(例如触发某个事件的时候执行它)

自执行函数

(创建函数)(传入参数);

((function(n){
    
})(100);

==(创建函数后立即执行)

((function(n){
    n = 100;
})(100);
