嵌套使用

插入图片

💡文件路径支持相对路径的书写形式:

​ 同级目录下:

![Pciture_a](/a.png)

锚点

[显示文本](#需要跳转到的标题)

💡标题这里更像是CSS语法，"#” 是锚点语法中的一部分，并不是Markdown中的"分级标题"语法, 因此不需要根据标题等级打出对应数量的”#"，只需要一个即可

示例:

[说明文档](#Readme)

...(中间段落)

## Readme

会被渲染为:

说明文档 /*可以看到链接标题的”(#Readme)“是不会被渲染的*/

…(中间段落)

Readme

🚧锚点并不支持含有括号的标题，即便是英文半角也不受支持，例如下标题:

标题a(含括号内容)

如果写成:

[标题a](#标题a(含括号内容))

>>> 标题a

明显可见识别内容将标题括号识别成了语法部分，因此即便渲染正确，却不一定能正确的链接和跳转

高亮

==待高亮文本==

>>> ==待高亮文本==

🚧高亮特性的支持并不具有普遍性，请在使用前确认编辑器的支持

上角标

需要上角标的字符^2^

💡"2"处可以是任意字符

渲染实例:

10^2^

>>> 10^2^

🚧上角标特性的支持并不具有普遍性，请在使用前确认编辑器的支持

下角标

需要上角标的字符~2~

💡"2"处可以是任意字符

渲染实例:

H~2~O

>>> H~2~O

🚧下角标特性的支持并不具有普遍性，请在使用前确认编辑器的支持
